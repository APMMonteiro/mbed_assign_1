# ESP32 Assignment 1

## Project Overview

This assignment develops an application that generates a pair of periodically repeating digital signals on an ESP32 microcontroller. The on/off state and nature of the signals are controlled via two push buttons. The application is implemented in both Arduino IDE and ESP-IDF on VSCode.

## Hardware Setup

### GPIO Pin Configuration

| Pin | Function | Purpose |
|-----|----------|---------|
| GPIO 22 | RED_OUTPUT_ENABLE_PIN | Enable/Disable button input |
| GPIO 18 | GREEN_OUTPUT_SELECT_PIN | Pattern selection button input |
| GPIO 32 | DATA_LED | Data signal output |
| GPIO 12 | SIGNAL_LED | Synchronization signal output |
| GPIO 25 | EXTRA_GND | Extra ground reference |

### Arduino IDE Build

Open [ESP32_Assignment_1/ESP32_Assignment_1.ino](ESP32_Assignment_1/ESP32_Assignment_1.ino) in the Arduino IDE and upload to your ESP32 board.

### Debug Mode Configuration

To enable debug mode with 1000x slower timing, enable the `Enable debug timing` option in the ESP-IDF menuconfig. Access the configuration editor by pressing `Ctrl+Shift+P` in VS Code and running:

```
> ESP-IDF: SDK Configuration Editor (Menuconfig)
```

Navigate to **Application Settings** and enable "Debug Timing":

![ESP-IDF Menuconfig - Debug Timing](docs/images/setting.png)
